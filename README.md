# Nyiur Nanggroe — AI-Powered Circular Economy Coconut Marketplace

Nyiur Nanggroe is a production-ready Web application connecting coconut farmers, local UMKM, and green buyers. Built to optimize Aceh's coconut economy, the platform uses AI-powered features, rich interactive dashboards, and complete e-commerce integrations to drive sustainable circular trade.

---

## Key Features

- **Circular Economy Marketplace** — Sell and buy coconut byproducts (cocopeat, coir, charcoal briquettes, VCO).
- **AI Visual Search & Chat** — Advanced AI assistant ("Nyai Nyiur") answering environmental queries, predicting price models, and identifying products from photos.
- **Eco-Impact Dashboard** — Real-time tracking of CO2 reduction and carbon emissions offset by coconut byproducts transactions.
- **Coconut Education Hub** — Interactive learning platform with quizzes and certification elements for local businesses.

---

## Tech Stack

- **Frontend Framework**: Next.js (App Router with Turbopack)
- **Database & Auth**: Supabase (PostgreSQL with Row Level Security)
- **AI Integrations**: OpenAI GPT & Google Gemini APIs
- **Design System**: Vanilla CSS, TailwindCSS, Framer Motion
- **Payments**: Midtrans Sandbox/Production Gateway

---

## Quick Start

### Installation

1. Clone the project and install:
   ```bash
   npm install
   ```
2. Setup local development configurations:
   ```bash
   cp .env.example .env.local
   ```
3. Run dev server:
   ```bash
   npm run dev
   ```
