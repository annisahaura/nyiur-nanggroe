# Nyiur Nanggroe — Future Roadmap

This document outlines the startup vision, features, and scale target phases for the Nyiur Nanggroe platform.

## Phase 1: Local Launch & Ecosystem Optimization ( Aceh Region )
- Hardening of security measures and performance optimizations.
- Onboarding farmers, local coconut co-ops, and UMKM groups.
- Visual AI search refinements for local coconut products and byproduct identifications.

## Phase 2: Mobile Applications ( Android & iOS )
- Wrap web application using React Native or Capacitor for native experience.
- Push notifications for order status, chats, and educational quizzes.
- Offline support for farmers with bad network connectivity.

## Phase 3: AI Price Prediction & Demand Forecasting
- Train predictive models based on historical coconut byproduct market values.
- Dashboard for sellers forecasting high demand periods for cocopeat/coir/briquettes.
- Carbon offset certificate issuance integrated into user purchase histories.

## Phase 4: National Expansion & Export Marketplace
- Scale database to handle 100,000+ monthly active users.
- Integration with international payment systems and shipping APIs.
- Multi-language support (Bahasa Indonesia & English).
